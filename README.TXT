My Link Saver - Умное хранилище ссылок

📋 Описание проекта

My Link Saver - это веб-приложение для сохранения и организации полезных ссылок. Проект состоит из двух микросервисов и фронтенда.

🏗️ Архитектура

- Auth Service: Микросервис авторизации (FastAPI + PostgreSQL)
- Main Service: Основной сервис для работы с ссылками (FastAPI + PostgreSQL)
- Frontend: Веб-интерфейс (HTML + CSS + JavaScript)

🚀 Быстрый старт

1. Клонируйте репозиторий:

git clone https://github.com/VapasiraStudio/Project_number1.git
cd Project_number1

2. Запустите все сервисы:

docker-compose up -d

3. Откройте в браузере:
   - Frontend: http://localhost:80
   - Auth Service API: http://localhost:8000/docs
   - Main Service API: http://localhost:8001/docs
